# SQL Interview Questions – 100 Solved PostgreSQL Problems

## 📌 Project Overview

This repository contains 100 SQL interview questions with solutions written in PostgreSQL. The questions range from beginner to advanced and cover concepts commonly asked in Data Analyst, Business Analyst, Data Science, and SQL Developer interviews.

## 🚀 Features

- 100 SQL interview questions with solutions
- Beginner to advanced SQL concepts
- PostgreSQL-compatible queries
- Window Functions
- Common Table Expressions (CTEs)
- Joins and Subqueries
- Ranking Functions
- RFM Analysis
- Pareto Analysis
- Dashboard Queries
